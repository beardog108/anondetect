Anonymity Network Detector

What is this?

AnonDetect is a simple script inspired by a test for social media logins.

This script detects if you have Tor, I2P, Freenet, or ZeroNet running on your system, by requesting images from their web interfaces. (Except for Tor Browser, which is leaked through resource:// URIs.)

This uses no server side code.

If you are using the Tor Browser Bundle, you are protected from this because localhost connections are blocked.

Identifying localhost/LAN web services using techniques similar to this is one of many ways websites can uniquely identify you, even when you are using a VPN service.
